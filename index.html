    Sobre

    Guia de bolso (e smartphone) p/ parte mais legal do FIFA: humilhar os adversários com dribles e comemorar os golaços.

    Comandos de drible assumem o jogador se movendo para cima da tela

    Comemorações Pro e EASFC precisam ser desbloqueadas (evoluindo seu jogador Pro e comprando na EASFC Catalogue) para serem usadas

    Comemorações marcadas como "agile players / jogador ágil" são executadas apenas pelos jogadores mais habilidosos

    LS

    Left stick, alavanca esquerda

    RS

    Right stick, alavanca direita

    LS ↑ / → / ↓ / ← / ...

    RS ↑ / → / ↓ / ← / ...

    Direção usando a alavanca direita

    DP ↑ / → / ↓ / ← / ...
